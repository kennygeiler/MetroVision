<action>
Add **`src/lib/boundary-fusion.ts`** exporting at minimum:

- `export type BoundaryFusionPolicy = "merge_flat" | "auxiliary_near_primary" | "pairwise_min_sources";`  
  - **`merge_flat`:** current behavior — concatenate + `clusterCutTimes` (delegate to existing `mergeInteriorCutSec` or inline equivalent) for backward compatibility.
  - **`auxiliary_near_primary`:** start from **primary** cut list clustered with ε; add an auxiliary time **only if** it falls in `(t-ε, t+ε)` around at least one **primary** cut after primary clustering (tunable: “attach to nearest primary”); then optional final ε-pass to dedupe.
  - **`pairwise_min_sources`:** (lighter variant of consensus) a cut at time `t` is kept only if **both** primary and auxiliary have a candidate within **ε/2** of `t` (implement via: for each auxiliary peak, check primary; merge centers) — document exact rule in file header.

Export:

`export function fuseBoundaryCutStreams(primarySec: number[], auxiliarySec: number[], policy: BoundaryFusionPolicy, epsSec?: number): number[]`  

If `epsSec` omitted, use **`boundaryMergeEpsilonSec()`**.

All inputs: finite, `>= 0`; filter non-finite before fusion. Output sorted ascending, suitable for **`cutsSec`** (interior instants only — caller supplies 0/duration if needed).
</action>
