<action>
Create **`scripts/detect-refine-fn-windows.ts`** invoked as:

`pnpm detect:refine-fn-windows -- <videoPath> --gold PATH --pred PATH [options]`

**Required flags:** `--gold`, `--pred` (eval JSON shapes as elsewhere).

**Options:**

- `--pad SEC` — half-width of each FN window (default **`2.0`**). Window film-absolute: `[max(0, t - pad), t + pad]`, clipped to optional global `--start` / `--end` if provided (same semantics as `detect-export-cuts`).
- `--detector adaptive|content` — passed through like `detect-export-cuts` (default **`adaptive`**).
- `--max-windows N` — process at most **N** FN times (in ascending gold order) to cap runtime (default **unlimited**).
- `--out PATH` — write merged predicted JSON (schema compatible with `eval:pipeline`: at least `{ cutsSec: number[] }`, include optional `refineMeta` object: `{ fnCentersSec, padSec, windowsRun, basePredPath, goldPath }`).

**Algorithm:**

1. Load gold/pred cuts via **`extractCutsSecFromEvalJson`**.
2. `ev = evalBoundaryCuts(gold, pred, tol)` with **`--tol`** (default **0.5**).
3. Let `fn = ev.unmatchedGoldSec` (already normalized).
4. Baseline interior cuts = `normalizeCutList` of pred cuts or raw pred — match **`evalBoundaryCuts`** expectation (use same list as eval input for pred).
5. For each selected FN center `t`, build window, call the **same** pipeline as `detect-export-cuts` for that window only: `prepareIngestTimelineAnalysisMedia` + `detectShotsForIngest` with `segmentFilmWindow` + offset splits, derive **`cutsSec`** for that window via same `splitsToCutsSec` logic (copy small private helper from detect-export-cuts or import if refactored).
6. Collect all **new** interior cut times from window runs that are **inside** the window and **not** already within **epsilon** of baseline (optional optimization — acceptable to merge everything).
7. **`mergeInteriorCutSec(baseline, collectedExtras)`** → final `cutsSec`.
8. Write `--out` JSON; print to stderr: baseline vs refined **P/R/F1** vs gold using **`evalBoundaryCuts`**.

Add **`package.json`** script: **`"detect:refine-fn-windows": "tsx scripts/detect-refine-fn-windows.ts"`**.
</action>
